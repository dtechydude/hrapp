Operational Flow

From Staff Onboarding to Payroll in Minutes

DYCONNECT enforces a structured workflow that ensures data integrity, approval gates, and permanent audit history at every stage.

Start Onboarding a Staff
01
Onboard Staff & Assign to Organization

Create a staff profile, upload documents, set grade level, bank details and deploy to a client organization or branch.

02
Configure Salary Structure

Define basic salary, allowances, and active deductions (PAYE, Pension, NHF, Loan, Cooperative, etc.) per staff record.

03
Capture Attendance & Leave

Record daily attendance, late arrivals, approved leave and overtime. All feed automatically into payroll computation.

04
Generate & Approve Payroll

Run monthly payroll — system computes gross, all deductions and net salary. HR Manager reviews and approves before locking.

05
Lock, Export & Archive

Payroll is locked (no further edits), salary slips are auto-generated, and the run becomes permanent historical data.

Access Control

9 Defined User Roles

Every user sees only what their role permits. Role-Based Access Control is enforced at every view, form and API endpoint.

System Administrator

Full access to all modules, user management, system configuration and audit logs.

HR Manager

Manages staff profiles, promotions, suspensions, leave approvals and employment history.

Accountant

Full access to accounting, general ledger, financial statements and payroll expense integration.

Payroll Officer

Creates and processes monthly payroll runs, manages deductions and exports salary schedules.

Branch Manager

Oversees staff within their branch, approves leave, tracks attendance and views branch reports.

Supervisor

Manages team attendance, submits leave requests and monitors deployment records.

Client Organization

Views deployed staff list, contract status and receives relevant notifications for their site.

Staff (Employee)

Views personal profile, payslips, leave balance, loan status and submits leave applications.

Auditor

Read-only access to audit trails, financial reports, payroll history and compliance records.

Enterprise Security

Built Secure From the Foundation Up

DYCONNECT-App follows industry security best practices, with additional layers of protection for sensitive payroll, financial, and personnel data. No shortcuts. No hardcoded secrets. No raw DATABASE.

Login Required on every protected view
Permission checks enforced at view level
CSRF protection on all forms
Best ORM — zero raw DATA
Secure file upload with type validation
Secrets via environment variables
Role-Based Access

Granular permissions ensure each user can only access their designated functions.

Immutable Records

Payroll history, employment events, and audit logs can never be deleted or overwritten.

Soft Delete

Records are never hard-deleted. Deactivation preserves full relational data integrity.

Password Hashing

All passwords are hashed using PBKDF2 algorithm. Never stored in plain text.
